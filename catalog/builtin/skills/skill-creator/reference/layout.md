# Skill folder reference

## Example tree

```text
skills/my-skill/
  SKILL.md           # required entry
  scripts/
    check.sh         # optional helper
  reference/
    api-notes.md     # optional deep dive
```

## Naming

- **skill-id**: ASCII lowercase, digits, hyphens (e.g. `deploy-staging`).
- Script names: descriptive, no spaces.
